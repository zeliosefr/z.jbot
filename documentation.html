Documentation de Z.J Bot

Explorez toutes les commandes slash pour gérer et modérer votre serveur Discord.

Bienvenue dans la Documentation de Z.J Bot

Z.J Bot est un bot de modération puissant et facile à utiliser, conçu pour aider les administrateurs de serveurs Discord à gérer et modérer leurs communautés. Découvrez ci-dessous toutes les commandes slash pour maintenir l'ordre sur votre serveur.

Installation

Pour ajouter Z.J Bot à votre serveur Discord, suivez les étapes suivantes :

  • 1. Cliquez sur le bouton ci-dessous pour accéder à la page d'ajout du bot.
  • 2. Connectez-vous à votre compte Discord si ce n'est pas déjà fait.
  • 3. Sélectionnez le serveur sur lequel vous souhaitez ajouter Z.J Bot.
  • 4. Cliquez sur "Autoriser" pour accorder les permissions nécessaires.
Ajouter Z.J Bot à votre serveur

Commandes Slash de Modération

Voici une liste des principales commandes slash de modération que vous pouvez utiliser avec Z.J Bot :

  • /kick [utilisateur] [raison] : Expulse un membre du serveur.
  • /ban [utilisateur] [raison] : Banni un membre du serveur. Utilisez cette commande pour les infractions graves.
  • /mute [utilisateur] [raison] : Réduit les capacités d'un membre (mute) en cas de comportement perturbateur.
  • /unmute [utilisateur] : Restaure les capacités d'un membre qui a été muté.
  • /warn [utilisateur] [raison] : Avertit un membre pour une infraction aux règles du serveur.
  • /clear [nombre] : Supprime un certain nombre de messages dans un canal textuel.
  • /lockdown [canal] : Verrouille un canal pour empêcher les membres d'y envoyer des messages.
  • /unlock [canal] : Déverrouille un canal pour que les membres puissent envoyer des messages à nouveau.
  • /report [utilisateur] [raison] : Signale un utilisateur pour un comportement inapproprié.

Les commandes slash sont faciles à utiliser et offrent une interface intuitive via Discord. Tapez simplement / et Z.J Bot vous proposera toutes les commandes disponibles !

Fonctionnalités Avancées

En plus des commandes de modération, Z.J Bot propose diverses fonctionnalités pour faciliter la gestion de votre serveur :

  • Logs de Modération : Gardez une trace des actions de modération exécutées sur votre serveur.
  • Protection anti-spam : Empêche les utilisateurs de spammer le chat avec des messages répétitifs ou indésirables.
  • Gestion des rôles : Gérez facilement les rôles et permissions des membres directement via des commandes.
  • Gestion des canaux : Créez, modifiez, et gérez les canaux de votre serveur de manière simple et rapide.
